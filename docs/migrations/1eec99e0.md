# Migration Guide: 1eec99e0 - flatten_dict Rename

## Overview
This is a non-breaking rename refactor for the `flatten_dict` function.

## Changes
- Function renamed from `flatten_dict` to `flatten_dictionary`
- All internal references updated
- Backward compatibility maintained via alias

## Migration Steps
No action required. The old name remains available as a deprecated alias:

```python
# Old (still works, deprecation warning)
from json2csv import flatten_dict

# New (recommended)
from json2csv import flatten_dictionary
```
